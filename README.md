# RPSSL
Dette projekt implementerer det udvidede spil “Rock–Paper–Scissors–Lizard–Spock” i C#.  
Programmet kører i konsollen og lader spilleren spille mod en computer-modstander, indtil en af parterne når 3 point.

**Funktionalitet**
- Spilleren kan vælge mellem: Rock, Paper, Scissors, Lizard, Spock  
- Agenten vælger tilfældigt  
- Kampreglerne er implementeret via matematiske forskelle mellem valgenes indeks  
- Først til 3 point vinder  

**Regler (kort opsummeret)**
Hver form slår to andre former:  
- Rock slår Scissors og Lizard  
- Paper slår Rock og Spock  
- Scissors slår Paper og Lizard  
- Lizard slår Paper og Spock  
- Spock slår Rock og Scissors
